About

The creator

The creator of this project is Avi Agola. Commonly known as Phytonking on Github.
He sees how the internet of things (IoT) is used in most day to day life and wanted to design a free service of open-source IoT-programming.


- "Iot is becoming a free space to output cretivity, bringing progress to internet techonology."

The system

The project has 3 components:

  • Device

    The device which could generate data and would be operated remotely.
    Coded in python. The device can send signals to the global server to the Cloud servers for the most up-to-date data.

  • Cloud Server

    This server operates independently and recieves and sends data to the device and interface server.

  • Interface Server

    This website is the interface server, where you can access your device's data and be able to send commands to your devices (Work in progress).
